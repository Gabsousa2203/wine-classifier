## Creacion del Ambiente Virtual

```bash
$ python -m venv venv

#Activar
$ venv\Scripts\activate

#Desactivar (opcional)
$ deactivate

#Instalar dependencias
$ pip install -r requirements.txt
```

## Correr la app

```bash
$ python model\wine_quality_model.py
```
